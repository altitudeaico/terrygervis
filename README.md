# Ascent Collection — Production Progress

Private progress preview for the Terry Jervis / Ascent Collection launch film.

## Publish as a live link (GitHub Pages)
1. Create a **new repository** — use an **unguessable name** (e.g. `ac-progress-9f3k2`), since GitHub Pages on a free plan is public.
2. Upload the contents of this folder (`index.html`, `README.md`, and the `assets/` folder) to the repo root.
3. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch → main / (root) → Save.**
4. Wait ~1 minute. Your live link will be: `https://<your-username>.github.io/<repo-name>/`
5. Share that link with Reginald.

To update progress later, replace the images in `assets/` and edit `index.html`, then push again.

Note: the page includes `noindex` so search engines won't list it, but anyone with the URL can view it. For true privacy, use a private repo with a preview host instead.
